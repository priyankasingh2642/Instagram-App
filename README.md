# INSTAGRAM app
A Flutter UI clone of Instagram app built to showcase frontend development, layout structuring, and design replication skills. This project focuses on creating a visually accurate and responsive interface using Flutter without any backend integration.

# What this project Demonstrates
Strong understanding of flutter UI widgets and layouts
Clean and modular code structure
Custom reusable components
Responsive design for different screen sizes
Ability to convert real world app design into flutter UI
Purely UI focused 

# Screens included
- Splash Screen
- Login & Signup Screen
- Home Feed UI with Stories
- Direct Messages(DM) Screen
- Search Screen
- Post Upload UI
- Reels UI
- Profile Screen with Tabs
